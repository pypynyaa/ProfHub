# Портал ИТ-профессий

Веб-портал для демонстрации и изучения различных профессий в сфере информационных технологий.

## Функциональность

### Общая часть (открытая)
- Каталог ИТ-профессий с подробными описаниями
- Информация о целях и задачах портала
- Регистрация новых пользователей
- Общая информация о портале

### Закрытая часть
- Панель администратора для управления контентом и пользователями
- Личный кабинет зарегистрированного пользователя
- Интерфейс эксперта для оценки профессий
- Система консультирования пользователей

## Технологии
- PHP 8.0+
- MySQL 8.0+
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Установка и запуск
1. Склонируйте репозиторий
2. Создайте базу данных MySQL
3. Настройте конфигурацию в `config/database.php`
4. Запустите миграции: `php migrations/migrate.php`
5. Запустите веб-сервер

## Структура проекта
```
/
├── assets/           # Статические файлы (CSS, JS, изображения)
├── config/           # Конфигурационные файлы
├── includes/         # PHP включаемые файлы
├── modules/          # Модули системы
├── templates/        # Шаблоны страниц
└── public/           # Публичная директория
```
