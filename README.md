# ProfHub - Платформа для онлайн-консультаций

## Описание
ProfHub - это веб-приложение для организации онлайн-консультаций между пользователями и консультантами. 
Платформа обеспечивает удобный интерфейс для общения в реальном времени.

## Основные функции
- Регистрация пользователей и консультантов
- Система аутентификации и авторизации
- Чат в реальном времени
- Управление статусами консультаций
- Просмотр истории консультаций

## Технологии
- Spring Boot
- Spring Security
- Thymeleaf
- H2 Database
- Bootstrap 5
- JavaScript

## Локальная установка и запуск
1. Клонируйте репозиторий
```bash
git clone https://github.com/pypynyaa/ProfHub.git
```

2. Перейдите в директорию проекта
```bash
cd ProfHub
```

3. Запустите приложение
```bash
./mvnw spring-boot:run
```

4. Откройте браузер и перейдите по адресу `http://localhost:8080`

## Развертывание на Railway.app

1. Создайте аккаунт на [Railway.app](https://railway.app)
2. Установите Railway CLI:
```bash
npm i -g @railway/cli
```

3. Войдите в свой аккаунт:
```bash
railway login
```

4. Создайте новый проект:
```bash
railway init
```

5. Разверните приложение:
```bash
railway up
```

## Переменные окружения

- `PORT` - порт для запуска приложения (по умолчанию: 8080)
- `DATABASE_URL` - URL базы данных (по умолчанию: jdbc:h2:mem:itportal)
- `DATABASE_USERNAME` - имя пользователя базы данных (по умолчанию: sa)
- `DATABASE_PASSWORD` - пароль базы данных (по умолчанию: пустой)
